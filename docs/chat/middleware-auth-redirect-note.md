# middleware 인증 리다이렉트 주석 처리 이력

## 파일 경로
- `src/lib/middleware.ts`

## 변경 내용
- 비로그인 사용자 접근 시 `/auth/login`으로 강제 리다이렉트하던 조건문 블록을 주석 처리함.

## 누가 처리했는지
- 이 변경은 **사용자(본인) 요청으로 주석 처리**한 사항임.

## 주석 처리 이유
- 기본 화면(`src/app/page.tsx`)이 먼저 보이도록 하기 위해 임시로 인증 강제 이동을 비활성화함.

## 원래 동작
- `user`가 없고 경로가 `/login`, `/auth`가 아니면 `/auth/login`으로 `redirect`.

## 복구 시 참고
- 인증 흐름을 다시 활성화하려면 주석 처리된 조건문/리다이렉트 블록을 복원하면 됨.
